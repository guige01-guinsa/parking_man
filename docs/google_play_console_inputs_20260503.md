## 2. 개인정보처리방침

```text
https://parking-man.onrender.com/privacy
```

사용자 지정 도메인 연결 후 사용할 수 있는 주소:

```text

```

현재 확인된 공개 주소는 `https://parking-man.onrender.com/privacy`입니다. `https://www.ka-part.com/privacy`는 사용자 지정 도메인이 해당 서비스로 연결된 뒤 사용하세요. Play Console에 넣기 전 브라우저에서 열리는지 확인하세요.

## 3. 앱 콘텐츠 입력 기준

데이터 보안:

```text
앱에서 개인정보 또는 사용자 데이터를 수집/처리함: 예
데이터 암호화 전송: 예, HTTPS 사용
사용자가 데이터 삭제를 요청할 수 있음: 예
광고 목적 데이터 공유: 아니요
```

수집/처리 데이터:

```text
개인 정보: 이름, 전화번호 또는 담당자 연락처가 차량대장/업무기록에 포함될 수 있음
사진 및 동영상: 번호판 촬영 이미지, 단속 사진, CCTV 요청 사진
앱 활동: 단속 기록, 차량 조회 기록, 업무 처리 기록
기기 또는 기타 ID: 일반적으로 수집하지 않음
위치정보: GPS 위치정보를 자동 수집하지 않음. 사용자가 입력한 단속 위치 또는 동/호/장소 정보만 처리
```

카메라 권한 사유:

```text
번호판 촬영과 단속 사진 등록을 위해 카메라 기능을 사용합니다. 촬영 이미지는 사용자가 단속 기록 저장 또는 업무 요청을 위해 제출한 경우에만 서버에 저장됩니다.
```

## 5. Play Console 진행 순서

1. 개발자 계정 신원 확인 완료 대기
2. 연락처 전화번호 인증 완료
3. 앱 대시보드에서 기본 스토어 등록정보 입력
4. 개인정보처리방침 URL 입력
5. 앱 콘텐츠의 데이터 보안 양식 작성
6. 앱 액세스 권한에서 테스트 계정 입력
7. 비공개 테스트 트랙 생성
8. AAB 업로드
9. 테스터 이메일 12명 이상 등록
10. 14일 이상 테스트 진행
11. 피드백 반영 후 프로덕션 출시 신청
